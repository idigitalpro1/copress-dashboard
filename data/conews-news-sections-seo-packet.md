# News Sections SEO Packet

Status: Buildout-ready planning packet
Goal: Give every town a consistent editorial spine that can scale without looking generic.

## Pillar Map

| Pillar | URL Pattern | Primary Intent | Internal Link Targets |
|---|---|---|---|
| Government | `/gov`, `/municipal-meetings` | agendas, budgets, hearings, policy | schools, features, LTO |
| Schools | `/schools`, `/sports/high-school` | board policy, safety, athletics | gov, sports, calendar |
| Shop Local | `/shop`, `/shop/{category}` | business discovery and sponsor revenue | events, visitor guide, features |
| Events | `/events`, `/calendar`, `/music` | what to do, meetings, local culture | shop local, visitor guide |
| Visitor Guide | `/visitor-guide`, `/outdoors` | tourism, activity guides, itineraries | shop local, events |
| Features | `/features`, `/investigations`, `/analysis` | long-form local reporting | gov, schools, LTO |
| Local Giving | `/community-giving`, `/go-fund-me` | verified campaigns and community support | events, features |
| Opinion | `/lto`, `/opinion` | letters and community voices | gov, schools |

## Article Template

Title:
`{Topic} in {Town}: {Date or Season}`

Meta description:
Use 155 to 160 characters, lead with the search action, and name the town.

Body:
1. Nut graf with local impact.
2. Verified source block.
3. Context/history paragraph.
4. What changes next.
5. Related links.
6. CTA: subscribe, submit, sponsor, or attend.

Schema:
- `Article` for stories.
- `Event` for calendar pages.
- `LocalBusiness` for directory profiles.
- `Organization` for network/town pages.

## Internal Linking Rules

- Every article gets 3 to 5 contextual internal links.
- One link goes to the town landing page.
- One link goes to a neighboring pillar.
- Shop Local links must be relevant and labeled if sponsored.
- LTO/opinion pages link back to the related civic topic.

## QA Checklist

- H1 matches intent.
- Title contains town.
- Meta description is unique.
- Images have descriptive alt text.
- No unsourced public claims.
- Sponsored content is marked.
- Canonical URL matches the correct host.
