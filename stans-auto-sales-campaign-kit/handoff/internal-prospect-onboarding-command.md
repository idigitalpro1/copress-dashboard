# Internal Prospect Onboarding Command

Visibility rule: internal operator prompt only. Do not link this from the client, advertiser, or owner-facing campaign kit page.

Use this command when building a reusable Top 20 prospect onboarding layer for an industry kit patterned after the Stan's Auto asset buildout.

## Prompt Command

Build the Top 20 prospect onboarding layer for this industry kit as an internal operator artifact only.

- Create or refresh 20 prospect slots for the target industry.
- For each prospect slot, generate three onboarding formats: email, SMS, and web preview link.
- Each onboarding link should point back to the full kit asset buildout, not to a client/owner page section.
- Required intake request: business name, website, logo, main phone, direct contact, best current offer, service area, 3-6 usable photos, and approval contact.
- Keep compliance language category-level until the advertiser verifies specific claims.
- Route generated assets through CoMSAT/SATCOM review before publication.
- Keep the owner-facing campaign page focused on assets, pricing, proof, kanban, handoff, and the shopping-cart info drawer.

## Owner-Facing Boundary

The owner-facing Stan's kit should not show prospect cards, prospect nav, or prospect onboarding copy. Owner-facing asset actions belong in the Marketing Kit drawer, where each asset can show placement price, CoMSAT/SATCOM route, open-asset action, and pricing estimate action.
