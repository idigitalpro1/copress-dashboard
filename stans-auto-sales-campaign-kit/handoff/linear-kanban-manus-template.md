# Linear / Kanban / Manus Template

Use this template to route a Stan's-style industry kit through SATCOM/CoMSAT.

## Linear Issue

Title:
Place advertiser content for `[Business Name]`

Description:
Build or place a sponsored content item using the industry kit pattern.

Fields:
- Business:
- Prospect ID:
- Industry:
- Placement type:
- Pricing route: `$50/mo - 10 posts` or `Immediate menu`
- Kit URL:
- Asset folder:
- Claim check owner:
- Publish lane: `5280.menu`, `Newsletter`, `Social`, `Banner`, `Radio`, `Landing`, `CRM`

Acceptance Criteria:
- Email onboarding link exists
- SMS onboarding link exists
- Web preview link exists
- Asset files are attached or linked
- Pricing route is selected
- SATCOM claim check complete
- Publish or schedule action recorded

## Manus Prompt

Create the advertiser content placement packet for `[Business Name]` using the Stan's Auto industry-kit pattern. Produce email, SMS, and web onboarding links; include the selected pricing route; attach or link all relevant assets; and create a kanban card with claim-check, approval, placement, and tracking steps. Do not publish specific inventory, pricing, financing, approval, or warranty claims unless the business verifies them.
